# example-e2e

The **separate** end-to-end automation suite for [`diffsense-ai/example-app`](https://github.com/diffsense-ai/example-app).

This is a live demo of [Testward](https://testward.app): when a PR in the app
repo changes a selector or route these specs depend on, Testward flags the
breakage on the PR — across repos, before CI.
