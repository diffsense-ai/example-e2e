# example-e2e

The **separate** end-to-end automation suite for [`diffsense-ai/example-app`](https://github.com/diffsense-ai/example-app).

This is a live demo of [DiffSense](https://diffsense.app): when a PR in the app
repo changes a selector or route these specs depend on, DiffSense flags the
breakage on the PR — across repos, before CI.
